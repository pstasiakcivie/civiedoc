## Preferences

The Preferences  section in CIVR allows users to personalize their dictation environment to match individual workflow styles, comfort, and system integration needs. These settings cover visual layout, dictation workflow, AI behavior, hardware support, and general application preferences.

### Preferences Configuration ‚Äì Summary Table

<Steps>
  <Step title="Steps to Open and Configure Preferences">
    Click the Settings tab and select Preference. Use the dropdown sections to expand each category (e.g., Editor, AI Features, Application Preferences).
  </Step>
</Steps>

**Section ‚Äî Details**

**Editor Preferences**\
Set your preferred background and field colors for the dictation window. Adjust line and paragraph spacing to improve readability. Change font size and picklist delimiters. Optionally, hide the right pane automatically during dictation.

**Dictation Workflow Settings**\
Set how many comparisons appear in reports. Enable auto-numbering for impressions. Automatically load original reports when adding addendums. Show a warning before signing reports. Enable auto-draft to save progress during dictation. Choose whether to show impressions at the top of the report.

**AI Features**\
Select the dictation engine (e.g., CIVR Custom Speech). Enable AI modes:

- Default Mode includes report correction and template matching.
- Turbo Mode auto-generates impressions and documents critical findings.
- Nitro Mode creates full reports from key findings.

**Hardware Support**\
Enable support for gaming mice. Configure external microphones. Set Push/Hold-to-Talk behavior if needed.

**Application Preferences**\
Use Safe Mode for a simplified version of CIVR. Disable the CIVR setup wizard if you don't want to see it again.

**Resetting Preferences**\
Click Reset All to quickly revert all settings to their original defaults.

<Tip>
  - Enable Auto Drafting to prevent accidental data loss during interruptions. A study can be accessed by only one user at a time; if the dictation window is closed without signing, the study will automatically move to Result Pending status.
  - Use Turbo Mode for a balance of automation and customization.
  - ‚ö†Ô∏è Disable Nitro Mode if full AI-driven reporting isn't needed.
</Tip>

## Dictation Mode

The Dictation Mode section helps you configure mic sensitivity, directionality, and noise cancellation for optimal voice recognition. Choosing the right recording mode ensures clear speech capture in different environments.

### Steps to Configure Dictation Mode

<Steps>
  <Step title="Click on the Settings tab, then select Dictation Mode.">
    
  </Step>
  <Step title="Choose the Beep Sound Volume (e.g., Volume 1) from the dropdown.">
    
  </Step>
  <Step title="Review and select the LED Color Mode based on your working environment: ">
    Blue: Quiet Environment (Default, recommended for most users).

    Green: Noisy Environment for Speech Recognition.

    Red: Noisy Environment for Classic Dictation. 

    Purple: Custom Mode (only configurable via SDK).
  </Step>
  <Step title="Adjust settings for each mode:">
     Set Directionality: OFF / HIGH 

    Set Noise Cancellation: OFF / MID / HIGH  

    Toggle Intelligent Auto Mode: ON / OFF
  </Step>
  <Step title="Fine-tune Directionality and Noise Cancellation using sliders at the bottom.">
    
  </Step>
  <Step title="Save">
    Click **Save** to apply or Reset to discard changes.
  </Step>
</Steps>
