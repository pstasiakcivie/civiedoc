## Manage Orders

The Manage Orders module allows users to efficiently view, track, and manage radiology orders and their associated reports. Users can filter, review, edit, link, reassign, or resend order-related data directly from this interface.

### Navigating to Manage Orders

<Steps>
  <Step title="Go to the top navigation bar.">
    
  </Step>
  <Step title="Click Support.">
    
  </Step>
  <Step title="From the dropdown, select Manage Orders.">
    
  </Step>
</Steps>

### Filters and Search Options

Located at the top, the filters allow users to narrow down results based on the following fields:

| Field                 | Description                                                      |
| --------------------- | ---------------------------------------------------------------- |
| Choose a Location     | Dropdown to filter orders by imaging location                    |
| Accession             | Search by unique accession number                                |
| Patient Name          | Search using patient‚Äôs full or partial name                      |
| Patient MRN           | Filter by Medical Record Number                                  |
| DOS (Date of Service) | Date range selector to specify the order timeframe               |
| Select a User         | Optional user filter to refine based on physician or radiologist |
| Filter/Clear          | Filter applies selections, Clear resets all fields               |

Each order row in the Manage Orders table includes a series of functional icons that allow users to interact with the report and manage its workflow. Below is a detailed breakdown of each icon's purpose from left to right:

**1. Report Icon** üìÑ

1. Clicking the Report icon opens the completed report of the selected order.
2. Upon hovering it displays a tooltip which shows the name of the physician who electronically signed the report.

**2. Action Icon üîó**

1. In the Manage Orders table, locate the completed order you want to link with pending studies. Click the Action icon in that row.
2. A Pending Open Orders popup will open, displaying a list of all pending studies available for linking.
3. In the popup, review the list of pending studies. Use the checkboxes next to each study to select one or more pending orders that should be linked to the completed order.
4. After selecting the appropriate studies, click the Link Report button. The selected pending studies are now linked to the chosen completed order, and their statuses are updated accordingly.

**3. Move Report to Another Accession Icon**

1. In the Manage Orders table, locate the row of the completed order you want to move. Click the Move Report to Another Accession icon.
2. Review the Information Prompt - The popup explains that the selected order‚Äôs report details will be copied to a new accession. The original order will be marked as Cancelled, and the new accession will be marked as Completed and Locked on the worklist.
3. In the Accession input field, type the accession number of the order to which the report should be moved.
4. After entering the accession number, click the Add button. The table below will populate with details such as: Accession, Procedure Description, Patient Name, Patient MRN and Ordering Physician.
5. Confirm that the target accession information matches the intended record (correct patient, procedure, etc.). Once confirmed, click the Apply button to complete the transfer.

**4. Delete/Dissociate Report Icon ‚ùå**

1. Upon clicking the Delete icon opens a confirmation popup to confirm deletion.
2. Selecting the Delete option on the pop-up moves the order back to the result pending.

**5. Resend ORU Icon üì§**

1. Resend ORU icon triggers a resend of the HL7 ORU message which retransmits the report data to downstream systems (e.g., EMR, PACS).

## Request Addendum

The Addendum Request feature in RadFlow allows users to formally assign a request for changes or additions to a previously signed radiology report. This is typically used when corrections, clarifications, or updates are needed after a report has been finalized.

**Assigned To**: Select the name of the physician or user responsible for reviewing the addendum.

**Reason**: Enter the reason for the addendum. This field is mandatory.

**Send Notification _(Optional)_:**

1. Check **Email** to send an email notification.
2. Check **Text** to send an SMS notification.

Click **Save** to submit the request or click **Cancel** to discard the request.
