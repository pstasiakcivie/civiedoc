The **Recently Dictated Studies** tab categorizes the studies worklist into five sections: **Result Pending**, **Signed Today**, **Drafts**, **Addendum Requested**, and **Today**. Each category provides specific actions for managing reports efficiently.

## Result Pending

These reports are awaiting final review before being signed.

<Steps>
  <Step title="Open dictation window">
    Click the **CIVR** icon or double-click the study to launch the dictation window and start dictating.
  </Step>
  <Step title="Sign the report">
    Upon completing dictation, click the **Sign** button to finalize the report.
  </Step>
</Steps>

## Signed Today

These reports have been signed within the current day.

<Steps>
  <Step title="View report">
    Click the **Report** icon to view the report.
  </Step>
  <Step title="Dictate addendum">
    Click the **CIVR** icon to dictate an addendum.
  </Step>
  <Step title="Confirmation prompt">
    A pop-up will confirm if the report is finalized before proceeding with the addendum.
  </Step>
  <Step title="Save as Addendum">
    If a user edits an already completed report and saves it again, it will be marked as **Addendum**.
  </Step>
</Steps>

The original report text and the addendum will be shown together for reference.

## Drafts

These are reports that have been dictated but not yet finalized.

<Steps>
  <Step title="View draft">
    Click the **Report** icon to view a draft report.
  </Step>
  <Step title="Delete draft">
    Click the **(X)** icon to delete a draft. Confirm the deletion in the pop-up window. This moves the study back to **Result Pending** status.
  </Step>
  <Step title="Resume dictation">
    Click the **CIVR** icon to resume dictation, then sign the report upon completion.
  </Step>
  <Step title="Bulk sign reports">
    To sign multiple reports at once, select the checkboxes beside the reports and click the **Sign** button in the top ribbon.
  </Step>
</Steps>

## Addendum Requested

These reports require modifications based on user feedback.

<Steps>
  <Step title="View report">
    Click the **Report** icon to view the report.
  </Step>
  <Step title="Dictate addendum">
    Click the **CIVR** icon or double-click the study to open the addendum dictation window.
  </Step>
  <Step title="Confirmation">
    A confirmation pop-up will appear before proceeding.
  </Step>
</Steps>

## Filtering the Worklist

All categories in the Recently Dictated Studies tab can be filtered based on criteria such as:

- User selection
- Location
- MRN (Medical Record Number)
- Accession Number
- Date range (Last hour, Last 4 hours, Today, Yesterday, Last 3 days, Last 7 days, Last month, etc.)

<Steps>
  <Step title="Apply filters">
    After entering filter criteria, click **Apply**.
  </Step>
  <Step title="Clear filters">
    Click **Clear** to reset all filters.
  </Step>
</Steps>

<Note>
  The CIVR worklist auto-refreshes every 1 minute to display the latest updates.
</Note>

## Right-Click Options

When a user right-clicks on a study in the Recently Dictated Worklist, a contextual menu appears with the following options:

| Option                | Description                                                             |
| --------------------- | ----------------------------------------------------------------------- |
| **View Event**        | Opens the event log or history associated with the selected study.      |
| **Copy**              | Copies the selected cell value to clipboard.                            |
| **Copy with Headers** | Copies the selected row along with column headers for context.          |
| **Paste**             | Enables pasting copied content into the selected row (if editable).     |
| **Export**            | Allows exporting study details in supported formats (e.g., CSV, Excel). |

## Grid & Column Settings

Users can customise how studies are displayed by adjusting the grid and column settings.

### Adjusting Grid Columns

<Steps>
  <Step title="Open column list">
    Click the **Columns** tab on the right side of the Recently Dictated Studies section.
  </Step>
  <Step title="Choose columns">
    A list of available columns will appear.
  </Step>
  <Step title="Select columns">
    Check or uncheck columns as needed to adjust the display.
  </Step>
</Steps>

### Customizing Grid Layout

1. Drag and drop columns to rearrange their order.
2. Resize columns by clicking and dragging the column edges.
3. Save custom layouts for personalized study management.

## Creating a Dummy Order

The **Dummy Order** feature enables users to explore and familiarise themselves with CIVR's dictation capabilities. It provides a safe environment to practice dictating and navigating the interface using a sample studyâ€”ideal for testing functionality without affecting real data.

### Steps to Create a Dummy Order

<Steps>
  <Step title="Click the Create Dummy Order icon">
    Click the **Create Dummy Order** icon.
  </Step>
  <Step title="Select site and CPT">
    A pop-up window appears. Select the **Site** and **CPT/Procedure**.
  </Step>
  <Step title="Generate the order">
    Click **OK** to generate the order.
  </Step>
  <Step title="Explore dictation window">
    The dictation window opens, allowing users to explore its features.
  </Step>
</Steps>

<Note>
  Since this is a dummy order, the **Sign** and **Draft** buttons are not present. Only the **Close** button is available to exit.
</Note>
