## Preview Worklist

In the Private and Shared Worklist, user can initially preview a worklist (all the records present) before starting it. To do so the user clicks the Preview option besides the required worklist.

The Preview screen displays a list of coding records that match the filter conditions set during worklist creation.

1. **Search Bar**
   - Use the Search Patient field at the top-left to quickly locate a coding record of the patient by ID or name.
2. **Filter and Refresh**
   - Use the Filter button to apply specific criteria to refine your search.
   - Following toggles are provided in the filter panel to filter by CAP status:
     1. **Is Back From CAP**
        - Displays records that were previously flagged for Coding Assistance (CAP) and have now been reviewed and returned by the physician.
     2. **Is Pending Question from CAP**
        - Filters records that are currently awaiting a response or clarification from the physician within the CAP portal.
   - Click the Refresh (âŸ³) button to reload updated data.
3. **Start Coding**
   - Click Start Worklist (top-right corner) to begin working on the records.
   - Else click the View icon besides the required coding record to start coding

## Mass Edit /Bulk Update

The Mass Edit (or Bulk Update) feature allows users to apply changes to multiple records simultaneously. This feature becomes available once multiple records are selected from the preview list.

### Steps to Perform a Mass Edit

1. **Select Records**: Use checkboxes to select multiple entries.
2. **Open the Edit Dialog**: Click the Edit button.
3. **Choose Fields to Update**:
   - Tags (Add)
   - Tags (Delete)
   - Assigned To
   - Coding Status
   - Modality

   Conditional Display â€“ Coding Status Field:
   - If all records are New/In Progress â€“ Approve appears.
   - If any records are Approved/Pending/Disabled â€“ Send Back to Coding appears.
4. **Enter New Value**
5. **Add Additional Fields (Optional)**
6. **Finalizing the Update**
   - Click Save or Cancel

## Marking a Record for Coding Assistance (CAP)

Use CAP when physician clarification is needed.

### Steps:

1. Click **CAP**
2. Select **CAP Reason**
3. (Optional) Add comments
4. Click **Submit**
5. Click **Cancel** to discard
