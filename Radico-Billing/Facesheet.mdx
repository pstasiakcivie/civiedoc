![1 Pn](/images/1.png)

**Accessing Facesheet**

<Steps>
  <Step title="To access the Order Entry interface:" icon="chevron-down">
    1. Navigate to the **Facesheet** tab from the left navigation menu.
    2. Use the **Search** bar to locate documents by **file name**.
    3. Click **Filter** to refine the results using one or more criteria:
       - Created Date Range
       - Client Name
       - File Name
       - Document Status
    4. Click **Export** to download the filtered list in **CSV **or **Excel **format.
    5. Click the **View** button (in the **Actions **column) to open a specific facesheet document.
  </Step>
</Steps>

## Add Facesheet Document

To add a facesheet document, click on **Add** button:

![2 Pn](/images/2.png)

<Steps>
  <Step title=" Select Client Location">
    **Client Location (Required)**: Use the dropdown to select the location associated with the facesheet.

    <img
      src="/images/f3.png"
      alt="F3 Pn"
      title="F3 Pn"
      className="mx-auto"
      style={{ width:"70%" }}
    />
  </Step>
  <Step title="Upload Document">
    **Attach Files**: Click **Attach Files** button to select and upload one or more .pdf facesheet files from your system.

    <img
      src="/images/f4.png"
      alt="F4 Pn"
      title="F4 Pn"
      className="mx-auto"
      style={{ width:"67%" }}
    />

    **Supported File Format**: Only .pdf format is allowed.

    **Upload Limits:** Maximum file size is **5 MB** per file, and you can upload up to **5 **files.
  </Step>
  <Step title="Manage Uploads">
    **Upload All**: Click this button to upload all the selected facesheet files in one go.

    **Remove All**: Clears the entire selection of files.

    **Remove Individual File**: Click the **X** icon next to any uploaded file to remove it from the list before uploading.
  </Step>
  <Step title="Save or Cancel">
    **Save**: Finalizes the upload and adds the facesheet.

    **Cancel (X)**: Closes the pop up without uploading or saving.
  </Step>
</Steps>

## **View and Edit Facesheet Records**

This section guides users through viewing, editing, and managing facesheet records in the RCM application.

![F6 Pn](/images/f6.png)

<Steps>
  <Step title="Open Facesheet">
    **To open a facesheet**: Click **View** from the **Actions menu (⋮)**  next to the desired record.

    ![F5 Pn](/images/f5.png)

    A split-screen interface appears:

    **Left Panel**: Displays the uploaded facesheet in PDF format.

    **Right Panel:** Displays all patient forms associated with the facesheet, such as **Patient Demographics, Insurance, Address**, and more. Enter the corresponding details as provided in the facesheet.

    <Note>
      **Document Viewer Note**:\
      If the PDF fails to load, a message appears: "_An error occurred while loading the document. Try reloading the page_."
    </Note>
  </Step>
  <Step title="Check Document Status">
    The current document status appears at the top of the right panel and updates automatically as form progresses, the status can be as follow:

    | Status          | Description                             |
    | :-------------- | :-------------------------------------- |
    | **New**         | Document has not been opened or edited. |
    | **In Progress** | Partially completed.                    |
    | **Completed**   | All required fields are filled.         |
    | **Invalid**     | Document has been marked as incorrect.  |

    To manually update the status:

    Click the **Actions** button at the top right. Choose **Mark as Complete** or **Mark as Invalid** based on the document's state.
  </Step>
  <Step title="Add Comments" stepNumber={3}>
    Click **Comments** to view any existing remarks for the facesheet.

    ![F7 Pn](/images/f7.png)

    Use the **Text Box **and**Attachment**  option to add new comments or upload related files.

    If there are no comments, the message **No comments found** will be displayed.
  </Step>
</Steps>

## **Add New Record**

![F8 Pn](/images/f8.png)

<Steps>
  <Step title="Review Patient Information">
    Review the facesheet PDF on the left panel and extract the required patient details for entering into the form.
  </Step>
  <Step title="Search for Existing Patient">
    Click **Search Patient **button at the top of the right panel.

    ![F9 Pn](/images/f9.png)

    Apply filters to search the required patient. If found, select the patient to auto-fill the form.

    ![F9 1 Pn](/images/f9.1.png)
  </Step>
  <Step title="Add New Patient (If Not Found)">
    Manually enter the following patient details from the facesheet:

    **Name, Date of Birth, Gender, SSN, Contact, Address, Employer Encounter Number **and **MRN**.

    ![F10 Pn](/images/f10.png)
  </Step>
  <Step title="Guarantor Details">
    If guarantor is**same as patient** , select the checkbox to auto-fill.

    If different enter **Name, DOB, Address, Phone, SSN, Gender, Employer,** and **Relationship**.

    ![F11 Pn](/images/f11.png)
  </Step>
  <Step title="Admission Information">
    Enter **Patient Class, Location, Place of Service, Admit Date,** **Discharge Dates**

    and **Referring Physician's Name**.

    ![F12 Pn](/images/f12.png)
  </Step>
  <Step title="Primary Insurance Information">
    Fill the details **Plan ID, Insurance Provider, Group Numbers, Insured Person’s Name, DOB, Address, SSN, Gender, Employer,** and **Relationship.**

    ![F13 Pn](/images/f13.png)

    <Note>
      **Note**

      Expand each section and fill all required fields. You may also add **Secondary or Tertiary** insurance if applicable.
    </Note>
  </Step>
  <Step title="Save Record">
    Click **Save** to store the entered data. The facesheet status changes to **In Progress**.
  </Step>
  <Step title="Mark as Complete">
    Once data entry is finished, select **Mark as Complete** from the **Actions** menu to change the status to **Completed**.

    ![F14 Pn](/images/f14.png)
  </Step>
  <Step title="Record-Level Actions">
    Each patient record includes the following options:

    **Edit**: Modify patient or insurance details.

    **Delete**: Remove the record (confirmation required).

    **Link**: Reassign the patient to another encounter or record.

    ![F15 Pn](/images/f15.png)
  </Step>
</Steps>

## **Link Patient to Facesheet**

This function ensures the correct patient is connected to the facesheet.

<Steps>
  <Step title="Click Link Patient">
    ![F16 Pn](/images/f16.png)

    Found next to the patient’s name in the record list. Opens the patient search interface.
  </Step>
  <Step title="Search and Select Patient">
    Use filters like Name, Last Name, MRN, or SSN. Click the arrow icon  <Icon icon="arrow-right-long" /> to select.

    ![F16 Pn](/f17.png)
  </Step>
  <Step title="Review and Update Details">
    The form auto-fills with patient data. Validate fields such as:

    - Name, DOB, Gender, SSN
    - Phone, Address, State, ZIP Code
    - Complete any required fields and click **Save** to link the patient.

    ![F18 Pn](/images/f18.png)

    <Note>
      **Note**: If required fields are missing, an error will appear. All data must be completed to successfully link the patient.
    </Note>
  </Step>
  <Step title="Attachment-Level Actions">
    Each document in the Attachments table includes:

    **View**: Opens the PDF viewer.

    **Download**: Downloads the document in original format.

    **Delete**: Removes the file (use only for incorrect uploads).

    <Tip>
      **Permissions Note**: Attachment actions may be limited to users with specific roles such as Admin or Coder.
    </Tip>
  </Step>
</Steps>

## **Add Patient (If patient not found)**

This page opens when a user clicks the **Add** button from the **Search Patient** page. Steps to add a new patient:

![F19 Pn](/images/f19.png)

<Steps>
  <Step title="Add Patient">
    Fill in the following details mentioned as per the facesheet document. Fields marked with asterisk (\*) are required/mandatory:

    Select the **Location** using the dropdown provided.

    Enter the **First Name, Last Name, DOB, Gender, ZIP Code, MRN, SSN, Phone Number, Address, City**, and **State**.

    ![F20 Pn](/images/f20.png)
  </Step>
  <Step title="Final Actions">
    **Save**: Click to add the patient and link them to the facesheet document.

    **Reset**: Clears the form.

  </Step>
</Steps>
