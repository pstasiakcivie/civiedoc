### Step-by-Step Guide

<Steps>
  <Step title="Navigate to the Users List Page">
    Click on **Users** from the left menu. The **Users List** page opens, displaying all available users.
  </Step>
  <Step title="Search for a User">
    Use the **Search Users** bar at the top-left to find a user by name or email.
  </Step>
  <Step title="Filter Users">
    Click the **Status** dropdown to filter users based on their status (**Active** or **Inactive**).
  </Step>
  <Step title="Reset Filters">
    Click the **Reset** button to clear any applied filters or search criteria and restore the full user list.
  </Step>
  <Step title="Export User Data">
    Click the **Export** button to download the user details in a structured format for record-keeping.
  </Step>
</Steps>

## Manage User Settings and Permissions

To disable a user or to manage user permissions, click the three-dots icon under the action column beside the user.

<Steps>
  <Step title="To Disable a User">
    Select the **Disable** option to deactivate the user account. The user will no longer have access to the system.
  </Step>
  <Step title="Manage User Permissions">
    Select the **User Permissions** option to open the **User Permission** settings. Assign or modify **Base Role Permissions**, **Organizations**, **Subscriptions**, and **Clients** as required.

    - Click **Submit** to save the changes.
  </Step>
</Steps>

## Add Users

<Steps>
  <Step title="Navigate to the Users List Page">
    Select the **User Management** tab from the left menu. It populates the list of existing users.
  </Step>
  <Step title="Open the Add User Screen">
    Click the **Add User** button in the top-right corner. A list of new users will be displayed.
  </Step>
  <Step title="Search and Select a User">
    Use the **Search Users** bar to find a specific user by name or email. Locate the user from the list and click the **Add User** button in the **Action** column.
  </Step>
  <Step title="Assign User Permissions">
    The **User Permission** screen will open after adding the user. Under **Base Role Permissions**, select the applicable roles such as **SuperAdmin**, **Coding Manager**, or **TestRole**.

    - Assign the user to relevant **Organizations**, **Subscriptions**, and **Clients**.
  </Step>
  <Step title="Save and Confirm">
    Click **Submit** to confirm the user addition or click **Cancel** if you do not wish to proceed.
  </Step>
</Steps>
