**Accessing the Worklist**

<Steps>
  <Step title="Navigate to the Worklist Section" icon="chevron-down">
    1. Click on the menu bar (≡) located on the top left corner of the screen.
    2. From the Menu options, select **Worklist**.\
       This will take you to the main Worklist dashboard, where you can create worklists.
  </Step>
</Steps>

## Creating a New Worklist

<Steps>
  <Step title="Enter Worklist Details">
    **Worklist Name**: Enter a name for the worklist in the Worklist Name field.

    **Select Category**: Choose the category for which you want to create the worklist.

    **Description (Optional)**: Provide a description in the Description field to define the worklist’s purpose.

    ![2025 07 07 12 26 38 Pn](/images/2025-07-07_12-26-38.png)

    - **Private or Shared**:

      To create a private worklist, check the checkbox (Mark as Private).

      To create a shared worklist, leave the checkbox unchecked.
  </Step>
  <Step title="Define Filter Conditions">
    Specify filter conditions to customize the worklist.

    Use **AND/OR** to combine operators with logical conditions.

    Apply comparison operators such as **Equals**, **Contains**, or **Greater Than** to refine your results.

    ![2025 07 07 12 36 04 Pn](/images/2025-07-07_12-36-04.png)
  </Step>
  <Step title="Preview Worklist">
    Click **Preview** to review the filtered results and ensure they meet the specified criteria.
  </Step>
  <Step title="Save Worklist">
    Click **Save** to save the worklist.
  </Step>
</Steps>

**Additional Steps for Shared Worklist**

<Info>
  ⚠️ Applicable only if you are creating a shared worklist (i.e., the "Mark as Private" checkbox is unchecked).
</Info>

<Steps>
  <Step title="Select User Group">
    A **User Group** section will appear.

    Choose from existing groups using the dropdown menu and select the corresponding users.

    ![2025 07 07 12 42 13 Pn](/images/2025-07-07_12-42-13.png)
  </Step>
  <Step title="Create a New User Group">
    Click the **Create User Group** button.

    Enter the **Group Name**, select users, and click **Submit**.

    ![2025 07 07 12 43 23 Pn](/images/2025-07-07_12-43-23.png)
  </Step>
  <Step title="Edit Existing User Group">
    Click the **Edit User Group** button.

    Select the group to be edited, add or remove users as needed, then click **Submit**.

    ![2025 07 07 12 44 16 Pn](/images/2025-07-07_12-44-16.png)
  </Step>
  <Step title="Allow Shared Users to Edit/Delete">
    Enable this option to grant editing and deletion privileges to shared users.
  </Step>
</Steps>

<Steps>
  <Step title="Save or Cancel" icon="chevron-down">
    **Save**: Finalise and store the worklist.

    **Clear**: Reset all entered data.

    **Cancel**: Discard all changes and return to the worklist view.
  </Step>
</Steps>
