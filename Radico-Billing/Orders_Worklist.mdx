## Creating a New Worklist

<Steps>
  <Step title="Enter Worklist Details">
    **Worklist Name**: Enter a name for the worklist in the **Worklist** **Name** field.

    The **Order Worklist** is selected by default in the **Category** field.

    **Description (Optional)**: Provide a description in the **Description** field to define the worklist’s purpose.

    **Private or Shared**:

    - To create a **Private** **Worklist**, check the checkbox (Mark as Private).

      To create a **Shared** **Worklist**, leave the checkbox unchecked.
  </Step>
  <Step title="Define Filter Conditions">
    Specify filter conditions to customize the worklist.

    Use **AND/OR Operators**  to logically combine the conditions.

    Apply comparison **Conditions** such as **Equals**, **Contains**, or **Greater Than** to refine your results.
  </Step>
  <Step title="Preview Worklist">
    Click **Preview** to review the filtered results and ensure they meet the specified criteria.
  </Step>
  <Step title="Save Worklist">
    Click **Save** to create the worklist for the selected category.
  </Step>
</Steps>

**Additional Steps for Shared Worklist**

<Info>
  Applicable only if you are creating a shared worklist (i.e., the **Mark as Private** checkbox is unchecked).
</Info>

<Steps>
  <Step title="Select User Group">
    A **User Group** section appears.

    Choose from existing groups using the dropdown provided and select the corresponding users.
  </Step>
  <Step title="Create a New User Group">
    Alternatively, Click the **Create User Group** button to create a new user group .

    Enter the **Group Name**, select **Users**, and click **Submit**.
  </Step>
  <Step title="Edit Existing User Group">
    Click the **Edit User Group** button.

    Select the group to be edited, add or remove users as needed, then click **Submit**.
  </Step>
  <Step title="Allow Shared Users to Edit/Delete">
    Enable the **Allow Shared Users to Edit/Delete** checkbox to grant editing and deletion access to shared users.

    <Note>
      Note:\
      The Shared Order Worklists include following default worklists-**Missing Ordering Doctors, Missing Modality, Missing Date of Service, Missing Reading Doctors**, and **Missing Reading Locations**. Each worklist highlights records with specific missing information.
    </Note>
  </Step>
</Steps>

<Steps>
  <Step title="Save or Cancel" icon="chevron-down">
    **Save**: Confirm and save the worklist.

    **Clear**: Remove all entered data and reset the form.

    **Cancel**: Exit without saving and return to the worklist view.
  </Step>
</Steps>

## Default Shared Worklists

The **Ordering Worklist** includes the following default shared worklists: **Missing Ordering Doctors, Missing Modality, Missing Date of Service, Missing Reading Doctors,** and **Missing Reading Locations**. Each worklist highlights records missing specific information.

 Click the **Preview** button next to any worklist to view its complete list of records. Use the **Filter** and **Refresh** options to refine the list as needed.

The interface is split into two panels:\
The **left panel** displays records with missing fields and provides input fields to fill in the data. \
The **right panel** shows the corresponding document or report, allowing you to review the content and update the missing details accordingly.

**Editing a Worklist**

To edit an existing private or shared worklist:

<Steps>
  <Step title="Open the Worklist for Editing">
    1. Click the **three-dot menu (︙)** next to the desired worklist name.
    2. Select **Edit** to make updates, if any changes are required.

    ![2025 07 08 15 16 19 Pn](/ow10.png)
  </Step>
  <Step title="Modify Worklist Name">
    Update the name to reflect any changes. Ensure the name remains descriptive and unique.

    ![Ow11 Pn](/images/ow11.png)
  </Step>
  <Step title="Update Description (Optional)">
    Edit or add information to define the worklist's scope or filtering criteria more clearly.
  </Step>
  <Step title="Change Privacy Settings">
    Toggle Mark as Private to restrict access to the current user only. Uncheck to allow sharing with other users.

    ![Ow12 Pn](/images/ow12.png)
  </Step>
  <Step title="User Group (For Shared Worklists Only)">
    This section becomes editable only when **Mark as Private** is unchecked.

    **Add Use**r: Search for and select users to include in the shared worklist.

    **Create/Edit User Group**: Organize users into groups for simplified sharing.

    **Allow Shared Users to Edit/Delete**: Enable this to grant edit or delete permissions to shared users.
  </Step>
  <Step title="Adjust Filter Conditions">
    Edit or add filter criteria by choosing a field (e.g., Approved At), an operator (e.g., Equals), and a value (e.g., Last 30 Days).

    Use **AND/OR** logic to combine multiple conditions as needed.
  </Step>
  <Step title="Preview Updated Records">
    Click Preview to view a list of records matching the updated criteria.
  </Step>
  <Step title="Save or Reset Changes">
    **Save**: Apply and store the updated changes.

    **Clear**: Discard all changes and revert to the original settings.
  </Step>
</Steps>

**Deleting Worklist**

<Steps>
  <Step title="Steps" icon="chevron-down">
    1. Click the **three-dot menu (…)** next to the worklist name you want to delete.
    2. Select **Delete** from the dropdown options.
    3. In the confirmation pop-up, click Yes to confirm and permanently delete the worklist.

    ![2025 07 08 15 16 19 Pn](/2025-07-08_15-16-19.png)

    <Info>
      **Note**: Deleted worklists cannot be recovered. Make sure to double-check before confirming.
    </Info>
  </Step>
</Steps>

**Hide Worklist**

<Steps>
  <Step title="Steps to Hide a Worklist" icon="chevron-down">
    1. Click the **three-dot menu (︙)** next to the shared worklist you want to hide.
    2. Select the **Hide toggle** to change its visibility status.
    3. The worklist will be hidden from the default view but is not deleted.

    ![2025 07 08 15 16 19 Pn](/2025-07-08_15-16-19.png)

    <Note>
      **Note:** Hidden worklists can be restored later using the appropriate filters or settings. No data is lost
    </Note>
  </Step>
</Steps>

**View Hidden Worklists**

<Steps>
  <Step title="Steps to View and Unhide a Worklist" icon="chevron-down">
    1. Click the **Show Hidden** toggle at the top of the worklist view.
    2. This will display all worklists that are currently hidden.
    3. You can then unhide any shared worklist by using the same **three-dot menu (︙)** and toggle off the Hide option.
  </Step>
</Steps>
