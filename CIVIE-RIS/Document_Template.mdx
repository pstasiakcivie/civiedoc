## **Adding New Document Template**

<AccordionGroup>
  <Accordion title=" Create Document ">
    Navigate to Document Templates Go to the **Settings** section and open the Document Templates module.

    <Steps>
      <Step title="Enter Document Details">
        Fill in the following details:\
        Click on the **Create Document** button located on the top-left panel.

        <img
          src="/images/documenttemplate2.png"
          alt="+newappointment Pn"
          title="+newappointment Pn"
          className="mx-auto"
          style={{ width:"83%" }}
        />

        **Document Name:** Enter the title for the new template.

        **Group/Tag:** Select the appropriate group (e.g., Online Forms, Other).

        **DICOMization**: Toggle ON/OFF based on whether the document should support DICOM format.

        **Active**: Ensure this toggle is enabled to make the template available for use.
      </Step>
      <Step title="Design Your Template">
        Use the editor toolbar to format the content:

        Add text, adjust fonts, apply bold/italic styles, align content, insert images, tables, or code blocks.

        You can also insert Parameters from the right-hand panel to auto-fill patient, order, or report data dynamically.
      </Step>
      <Step title="Configure Document Settings (Optional)">
        Expand the Document Settings panel on the right:

        Set the Form Type, Delivery Format, Language, and Location if applicable.

        Apply Rule Settings if any conditions or workflows are to be attached.
      </Step>
      <Step title="Finalize the Template">
        Once completed, click the **Submit** button to finalize the template.
      </Step>
    </Steps>
  </Accordion>
</AccordionGroup>

## **Editing Document Templates**

<Accordion title="Document Templates">
  <Steps>
    <Step title="Access Document Templates">
      Navigate to the **Settings** section and open the **Document Templates** module.

      <img
        src="/images/documenttemplate3.png"
        alt="+newappointment Pn"
        title="+newappointment Pn"
        className="mx-auto"
      />
    </Step>
    <Step title="Select a Template to Edit">
      Click the **Edit** icon next to the chosen template to open it
    </Step>
    <Step title="Make Necessary Changes">
      Update the document content or formatting as needed using the editor tools.
    </Step>
    <Step title="Save the Updates">
      Once all edits are complete, Click the **Submit** button to save your changes.
    </Step>
  </Steps>
</Accordion>

## **Deleting Document Templates**

<Accordion title="Deleting Document ">
  <Steps>
    <Step title="Open Document Templates">
      Go to the **Settings** section and select the **Document Templates** module.

      <img
        src="/images/documenttemplate4.png"
        alt="+newappointment Pn"
        title="+newappointment Pn"
        className="mx-auto"
        style={{ width:"90%" }}
      />
    </Step>
    <Step title="Find the Template to Delete">
      Browse or search for the document template you wish to remove from the list.
    </Step>
    <Step title="Click the Delete Icon">
      Click the **Delete** icon next to the selected template.
    </Step>
    <Step title="Confirm Deletion">
      A confirmation prompt will appear. Click **Yes** or **Confirm** to permanently delete the template
    </Step>
  </Steps>
</Accordion>
