# **Adding a New Request**

To submit a new medical record request: Click on the **\+ New Request** button at the top right.

![Addnew MRR Pn](/images/addnewMRR.png)

An **Add Request** form will open, this form allows you to initiate a new request for a patient‚Äôs medical records, including images and reports, based on selected parameters as follows:

<AccordionGroup>
  <Accordion title="Request Details Tab">
    <Steps>
      <Step title="Select Request Details tab">
        ![Addreq Pn](/images/Addreq.png)

        **Location**: Choose the relevant facility from the dropdown.

        **Due Date:** Enter the expected completion date.

        **Request Type**: Select the priority type (e.g., STAT VERBAL, Routine).

        **Status:** Default is set to NEW, but can be updated to Processing, Completed, etc.
      </Step>
      <Step title="Requested By">
        **Requested By:** Choose the name of the requestor from the dropdown. This could be a physician or internal user.
      </Step>
      <Step title="Record Belongs To">
        ![Recordbelongto Pn](/images/recordbelongto.png)

        **Search Patient**: Start typing the patient‚Äôs name to auto-search and link the request to a patient profile.
      </Step>
      <Step title="What is Requested">
        Select one or both options: **Reports** and **CD**. Specify the Exam Duration using: **Start Date** and **End Date**.
      </Step>
      <Step title="Type of Studies">
        Use the dropdown to Search or Select Studies (e.g., MRI, CT, Mammogram).

        ![Typeofstudy Pn](/images/Typeofstudy.png)

        <Note>
          **Note**: You must select at least one modality to proceed.
        </Note>
      </Step>
      <Step title="Please indicate How You Would Like These to Be Received">
        ![Etf Pn](/images/etf.png)

        Choose between: **Other (Email/Fax/Mail/Text)** or **In-person Pickup**

        If **Other** is selected Add one or more delivery methods:

        **Delivery Method**: Select Email, Mail Address, Text, etc.

        **Delivery To**: Specify the destination (e.g., email address or mailing address).

        **Contact Name**: Enter the recipient‚Äôs name.

        Use the **\+** icon to add multiple methods and delete to remove.

        If **Inperson** selected Add pickup details:

        ![Inperson Pn](/inperson.png)

        Enter **Pickup Date Time**. Enable checkbox if it is **Same as Patient**. Provide the Pickup Person Name and Phone details.
      </Step>
      <Step title="Save and Cancel">
        Click **Save** to submit the request or click **Cancel** to exit without saving.
      </Step>
    </Steps>
  </Accordion>
  <Accordion title="Comments Tab">
    Switch to the **Comments tab** to add comments or add notes for tracking purposes.

    <img
      src="/images/COMMENTSMRR.png"
      alt="COMMENTSMRR Pn"
      className="mx-auto"
      style={{ width:"91%" }}
    />

    To add a comment: Click **Add Comment**. Enter the comment/Message in the text box provided. Click the **Add** button to post the comment.
  </Accordion>
  <Accordion title="Documents Tab">
    <Steps>
      <Step title="Choose Fax Document">
        Navigate to the **Document** tab to view documents received via fax. Click on **Choose Fax Document** to open a new window, where you can search by **File Name**, **Patient Name**, or **Create Date**.

        ![Documenttabfax Pn](/images/documenttabfax.png)

        Use the dropdown to filter by document status (e.g., New, Reviewed). Click **Select** next to the relevant file to attach it to the current request.

        <img
          src="/images/documenttabfax2.png"
          alt="Documenttabfax2 Pn"
          title="Documenttabfax2 Pn"
          className="mx-auto"
          style={{ width:"85%" }}
        />
      </Step>
      <Step title="Scan Document">
        This allows users to scan physical documents directly into the system and attach them to the request.
      </Step>
      <Step title="Upload From Computer">
        Use this option to upload digital files (PDFs, images, etc.) from your local system.
      </Step>
      <Step title="Save and Cancel">
        Click **Save** to apply changes or Click **Cancel** to exit without saving.
      </Step>
    </Steps>
  </Accordion>
</AccordionGroup>

# Editing a Medical Record Request

To **view and edit existing requests**, navigate to the **Medical Record Request** landing page.

In the last column of each row, you'll find an **Edit icon (‚úèÔ∏è)**. Click it to open the corresponding request. You can then **review and update** the information as required.

**Steps to Edit a Medical Record Request**

<AccordionGroup>
  <Accordion title="Request details Tab">
    <Steps>
      <Step title="Request details">
        Click the **Edit** icon (‚úèÔ∏è) from the Medical Record Requests list.

        A right-side panel titled **Edit** **Request** will open displaying detailed request information.
      </Step>
      <Step title="Verify Request ">
        View the Created By, Created Date, Modified By, and Modified Date at the top.

        This helps in tracking who initiated or last updated the request.
      </Step>
      <Step title="Review & Update Request Details">
        ![Edit1 Pn](/images/Edit1.png)

        **Location**: Select the applicable facility from the dropdown.

        **Due Date**: Adjust the due date if necessary.

        **Request Type**: Choose from available types such as STAT or Routine.

        **Status**: Update the request status (e.g., NEW, PROCESSING, COMPLETED).
      </Step>
      <Step title="Referring Physician Details">
        Confirm or update the Referring Physician name. Phone and fax numbers are auto populated if linked.

        ![Reqbyedit Pn](/images/reqbyedit.png)
      </Step>
      <Step title="Patient Information Section">
        Review details under Patient Information including: **Patient Name**, **Gender**, **Date of Birth**, **Primary Phone**, and **Address**.

        ![Recordbelongtoedit Pn](/images/recordbelongtoedit.png)

        Click the **Eye** icon to view more patient information or use the Trash icon to remove the record.
      </Step>
      <Step title="Select What is Requested">
        Tick the checkboxes for: Reports or CD this defines what part of the medical record is being requested.

        ![C Dedit Pn](/images/CDedit.png)
      </Step>
      <Step title="Set the Exam Duration">
        Enter the Start Date and End Date for the requested exams.
      </Step>
      <Step title="Type of Studies">
        Use the Search or Select Studies dropdown to choose the specific study type (e.g., MRI, CT, Mammogram).

        ![Typeofstudyedit Pn](/images/Typeofstudyedit.png)
      </Step>
      <Step title="Entering the Note">
        Update your note or instruction. Clearly specify: 

        Purpose of the note (e.g., missing consent, follow-up required). Responsible person/team if action is needed.

        ![Notesedit Pn](/images/notesedit.png)

        Any deadlines or critical dates if applicable.
      </Step>
      <Step title="Adding Delivery Methods">
        **Select the Delivery Method**

        You can choose between two main options: Other (Email / Fax / Mail / Text) or In-Person Pickup

        ![Etfedit Pn](/images/etfedit.png)

        By default, the Other option is selected. You may click the \+ icon to add another delivery method, or the delete icon to remove a specific method.

        Once basic request details are updated, you can access additional tabs for more context or required documentation.
      </Step>
    </Steps>
  </Accordion>
  <Accordion title="Procedures Tab">
    <Steps>
      <Step title="Link More Exams">

      </Step>
    </Steps>
  </Accordion>
  <Accordion title="Comments Tab">

<Note>
  **Best Practices**

  - Always double-check patient information and referring details before submitting.
  - Use STAT type only for urgent requests.
  - Keep track of changes via the Modified Date and Modified By fields for auditing.
